Este proyecto fue hecho con el xhampp: 
1. Para ejecutarlo se debe copiar en el adirectorio de proyecto_biblioteca en la carpeta
donde va se hosteado, que en este caso es la siguiente ruta C:\xampp\htdocs\
2. Pegar el directorio php_library en la siguiente ruta C:\xampp\mysql\data
3. Ejecutar el XHAMPP y activar el apache y el mysql.
4. copiar y pegar la siguiente ruta para el navegador.  http://localhost/Biblioteca/proyecto_biblioteca/index.php
